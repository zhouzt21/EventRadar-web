# EventRadar Website

This folder contains the standalone EventRadar project website.

Deployment: https://zhouzt21.github.io/EventRadar-web/

Open `index.html` directly in a browser to view the page. All required website assets are kept inside this folder:

- `static/css/`: EventRadar-specific styles
- `static/images/`: exported figures and video poster
- `static/videos/`: demo video
- `static/vendor/`: local third-party CSS dependency
- `docs/`: source notes and PDF figures
